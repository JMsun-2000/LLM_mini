# LLM_mini

先把「3 个月实战大纲」定死（我们就按这个走）

# 总目标
从原理懂 → 代码会 → 能训练 → 能微调 → 能部署 → 能写简历
# 第 1 个月：手写小 GPT，吃透底层（最关键）
月 1 目标：不用任何高级库，自己用 PyTorch 写出 最小可用 GPT，从 0 到生成文本。
- 周 1：PyTorch 核心 + 张量 / CUDA / 数据加载
- 周 2：手撸 Attention + Mask + LayerNorm + FFN
- 周 3：堆叠 Transformer Block，完成模型结构
- 周 4：写训练循环 + 自回归生成，跑通一个小模型
# 第 2 个月：开源大模型实战（Llama / Qwen / 微调）
月 2 目标：能跑、能微、能部署，做出能展示的项目。
- 周 1：跑通开源模型推理 + 生成原理
- 周 2：LoRA / QLoRA 微调原理 + 实战
- 周 3：INT4/INT8 量化 + vLLM 加速
- 周 4：做一个对话 / 问答小系统
# 第 3 个月：工程化 + 分布式 + RAG（简历级项目）
月 3 目标：像大厂工程师一样，做完整可落地系统。
- 周 1：数据清洗、分词、构建训练语料
- 周 2：多卡训练 DDP / FSDP 实战
- 周 3：RAG 检索增强生成全套
- 周 4：完整项目打包 → 简历可写
# 我们的玩法规则（超级简单）
你每次发我：
第 1 月・周 1
或
第 1 月・周 2 开始

带你一步步写
你哪里不懂我讲到懂
你有空就来，没空就停，完全按你的节奏。
