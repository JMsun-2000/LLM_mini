# Day1：PyTorch 环境搭建 + 张量（Tensor）核心操作
核心原理（只讲关键逻辑）
1. PyTorch 是动态图框架，计算过程实时构建，适合深度学习快速迭代，是手撸大模型的首选
2. 张量是 PyTorch 的核心数据结构，本质是多维数组，所有模型计算都基于张量，与 Numpy 数组互通但支持 GPU 加速
3. 张量的核心属性：shape（维度）、dtype（数据类型）、device（存储设备：CPU/GPU）

今日练习（必做，检验是否真懂）

3. 实现两个 2×3 张量的逐元素乘法和矩阵乘法（注意矩阵乘法的维度匹配要求）


# Day2：张量的梯度计算（autograd）+ 求导实战
核心原理（只讲关键逻辑）
1. 大模型训练的核心是反向传播求梯度，PyTorch 的autograd模块自动完成梯度计算，无需手动推导
2. 张量设置requires_grad=True时，会追踪所有对它的操作，调用.backward()会自动计算梯度，梯度存于.grad属性
3. torch.no_grad()：上下文管理器，禁用梯度追踪，用于推理阶段，节省内存

今日练习（必做，检验是否真懂）
1. 定义函数y = 3x³ + 2x² - 5x + 1，用 PyTorch 求 x=1 时的导数dy/dx
2. 创建一个 2×2 的随机权重张量w，构建损失函数loss = (w @ w).sum()，实现两次反向传播，并在每次传播前做梯度清零，观察梯度变化
